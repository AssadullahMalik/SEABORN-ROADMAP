# 📊 Seaborn Complete Guide & Mini Project 🚀

This repository documents my complete learning of **Seaborn**, a Python statistical visualization library, and includes a **mini project** demonstrating real-world data visualization.

---

## 📌 About Seaborn
Seaborn is built on top of **Matplotlib** and provides:
- Cleaner, high-level interface for creating plots
- Built-in themes and color palettes
- Functions for statistical visualizations like distributions, correlations, and regressions  

It’s widely used in **Data Science, Machine Learning, and AI**.

### 📊 Distribution Plots

- Visualize data spread and distribution

### 📈 Line Plot
- Show trends over time (`lineplot`)

### 📊 Bar Plot
- Compare categories (`barplot`)

### 🔵 Scatter Plot
- Explore relationships (`scatterplot`)

### 🎨 Box Plot
- Detect median, spread, and outliers (`boxplot`)

### 🔥 Heatmap
- Correlation matrices (`heatmap`)

### 🧠 Pair Plot
- Show pairwise relationships between all numeric columns (`pairplot`)

### 🎭 Styling & Themes
- Themes: `darkgrid`, `whitegrid`, `dark`, `white`, `ticks`
- Color palettes: `Set1`, `Set2`, `viridis`, etc.
